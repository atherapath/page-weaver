Green & Black — Responsive Layout

Side-by-side content

This layout places an image and a text section side by side on wider screens and stacks them vertically on smaller screens using a CSS @media query. The design emphasizes green accents on a black background for a bold, high-contrast look.

Resize the window to see the blocks change from a two-column layout to a single-column stack. The visual elements use subtle borders and gradients to keep the theme cohesive.
